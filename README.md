# 💻 Laptop Request Catalog Items

This project is a **ServiceNow Catalog Item** that streamlines the process of requesting laptops within an organization. It eliminates manual delays by providing a digital, dynamic form with approvals, tracking, and documentation.  

---

## 📖 Project Description

**Category:** ServiceNow System Administrator  
**Skills Required:** UIPath RPA, Tanzu Application Service  

### Problem Statement
Employees in the organization need a quick and efficient way to request laptops for work. The current process is manual and prone to delays, with no dynamic form behavior to guide users or ensure accurate data collection.  

To address this, a **Service Catalog Item** has been created in **ServiceNow**, allowing employees to easily request a laptop with:  
- Dynamic fields  
- Clear instructions  
- Reset option for resubmission  
- Approval workflows  
- Governance tracking and deployment readiness  

---

## 🚀 Features
- User-friendly request form for laptops  
- Dynamic fields based on user choices  
- Option to reset and re-submit form  
- Automated approval workflow  
- Request tracking for governance  
- Scalable for different laptop models/specifications  

---

## 🛠️ Technologies Used
- **ServiceNow** (Catalog Items, Client Scripts, Workflows, UI Policies)  
- **JavaScript** (for form logic and dynamic behavior)  
- **GlideRecord** (for backend server-side logic)  
- **UIPath RPA** (for process automation)  
- **Tanzu Application Service** (for deployment/management)
